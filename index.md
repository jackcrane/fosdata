# fosdata docs

This is a documentation site for the [speegled/fosdata](https://github.com/speegled/fosdata) package.

## Updating the documentation

In R, import an updated version of fosdata, then run the src/generate.r script. This will generate a fresh fosdata.json file, then use src/parse.js to generate the md files for the actual documentation.

## Sample files

Sample files are stored in the src/samples folder. They are automatically matched by file name to the dataset's name.
